# Selasar---Sistem-Informasi-Manajemen-Event-Kampus
Selasar merupakan sistem informasi manajemen event kampus berbasis web yang dikembangkan menggunakan Laravel 12 dan Filament Admin Panel. Aplikasi ini dirancang untuk memudahkan mahasiswa dalam menemukan, mendaftar, dan mengikuti berbagai kegiatan kampus secara terpusat, mulai dari seminar, workshop, lomba, pelatihan, hingga kegiatan organisasi.
